# MMO Packman Prototype

`docs/game_design.md` をベースにした、協力型 MMO Pac-Man の **Playable MVP** です。

## クイックスタート

```bash
npm install
npm run dev
```

- Client: `http://localhost:5173`
- Server: `http://localhost:8080`

## スクリプト

- `npm run dev` クライアント + サーバー同時起動
- `npm run check` 型チェック
- `npm run build` 本番ビルド
- `npm run start` ビルド済み成果物で起動
- `npm run simulate` AI-onlyシミュレーション（x2 / x5）
