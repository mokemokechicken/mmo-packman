# Design: spectator-free-camera-minimap

## Approach
1. 観戦カメラ状態（`follow` / `free`、ズーム倍率、自由カメラ中心座標）をクライアント状態として管理する。
2. 描画時のカメラ中心算出を拡張し、追従モードでは既存挙動を維持、自由モードでは手動中心座標を採用する。
3. 観戦UIに以下を追加する。
   - モード切替ボタン（追従/自由）
   - ズーム操作（ボタン + ホイール）
   - 俯瞰ミニマップ（プレイヤー位置 + 現在ビューポート）
4. ミニマップクリック時は最寄りプレイヤーへのフォーカスを優先し、該当なしなら自由カメラ中心をクリック位置へ移動する。

## Validation
